# Personal dotfiles

Zsh configuration with Powerlevel10k, antidote plugin manager, and fzf.

## Install

```bash
git clone https://github.com/mkocabas/dotfiles.git ~/dotfiles
cd ~/dotfiles
./install.sh
```

## What's included

- **Zsh plugins**: autosuggestions, syntax highlighting, completions, git
- **Prompt**: Powerlevel10k (lean theme)
- **Tools**: fzf, eza aliases, tmux shortcuts
- **Aliases**: Safe file operations, macOS helpers, utility functions
